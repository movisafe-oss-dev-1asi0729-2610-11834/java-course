# Checklist de rúbrica

## C01. Course Structure & Flow

- [ ] Las lecciones avanzan de lo básico a lo más complejo.
- [ ] Cada video se conecta con el anterior.
- [ ] La duración total se mantiene cerca de 1 hora.
- [ ] El curso está pensado para estudiantes de secundaria sin experiencia previa.

## C02. Technical Accuracy

- [ ] Todos los programas de Java fueron probados.
- [ ] Los ejemplos compilan correctamente.
- [ ] Las explicaciones usan conceptos correctos.
- [ ] Se evitan prácticas confusas para principiantes.

## C03. Target Audience Engagement

- [ ] El lenguaje es simple y directo.
- [ ] Se usan ejemplos cotidianos.
- [ ] Hay pausas para practicar.
- [ ] No se usa jerga innecesaria.

## C04. Video Production Quality

- [ ] El audio se escucha claro.
- [ ] El texto del editor se puede leer fácilmente.
- [ ] La grabación de pantalla no está borrosa.
- [ ] El video se subió a YouTube como no listado.

## C05. Course Outline Document

- [ ] El README sigue la estructura del Anexo A.
- [ ] Están los enlaces de YouTube.
- [ ] Está el enlace del repositorio de GitHub.
- [ ] Están los enlaces a editores online.
- [ ] Está el resumen de elaboración en equipo.
