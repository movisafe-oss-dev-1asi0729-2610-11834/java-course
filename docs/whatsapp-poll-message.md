# Mensaje para encuesta de WhatsApp

Chicos, para el trabajo de Responsabilidad Social debemos crear un curso básico de Java para estudiantes de secundaria.

Cada integrante debe escoger una lección. Cada uno será responsable de:

- Grabar 1 video tutorial.
- Crear su código inicial y código final.
- Subir su aporte al repositorio de GitHub.
- Preparar su parte del README siguiendo la estructura del Anexo A.
- Usar una herramienta online como Replit, JDoodle u OnlineGDB.

Opciones de lección:

1. Lección 1: Introducción a Java y primer programa “Hola Mundo”.
2. Lección 2: Variables, tipos de datos, operadores y expresiones.
3. Lección 3: Condicionales y bucles: if, else, for y while.
4. Lección 4: Métodos, parámetros y entrada de datos con Scanner.
5. Lección 5: Programación Orientada a Objetos: clases, objetos, constructores, encapsulación y mini proyecto final.

Voten por la lección que quieren trabajar. La idea es que nadie repita tema para completar el curso completo.
