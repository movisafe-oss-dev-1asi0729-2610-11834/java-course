# Guion de video - Lección 1: ¿Qué es Java y la programación?

**Duración sugerida**: 8 a 10 minutos  
**Herramienta**: Replit, JDoodle u OnlineGDB  
**Objetivo**: Que el estudiante entienda qué es programar, qué es Java y ejecute su primer programa.

## Estructura del video

### 0:00 - 1:00 | Bienvenida
Hola. En esta lección aprenderás qué significa programar y ejecutarás tu primer programa en Java. No necesitas instalar nada, solo usarás tu navegador.

### 1:00 - 3:00 | ¿Qué es programar?
Explica que programar es escribir instrucciones para que una computadora realice una tarea. Usa un ejemplo cotidiano: una receta, una lista de pasos o instrucciones para preparar una bebida.

### 3:00 - 4:30 | ¿Qué es Java?
Java es un lenguaje de programación usado para crear aplicaciones. En este curso se usará para aprender conceptos básicos de forma ordenada.

### 4:30 - 7:30 | Primer programa
Abrir un editor online y escribir:

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Hola mundo desde Java!");
    }
}
```

Explica brevemente:
- `class Main`: nombre principal del programa.
- `main`: punto de inicio.
- `System.out.println`: muestra un mensaje en pantalla.

### 7:30 - 9:00 | Práctica rápida
Pide al estudiante cambiar el mensaje por su nombre o una frase corta.

### 9:00 - 10:00 | Cierre
Resume que Java ejecuta instrucciones en orden y que en la siguiente lección aprenderán a guardar datos usando variables.

## Checklist de grabación

- Aumentar el zoom del editor.
- Ejecutar el programa antes de terminar.
- Usar lenguaje simple.
- Evitar mencionar nombres de autores o equipo dentro del código.
