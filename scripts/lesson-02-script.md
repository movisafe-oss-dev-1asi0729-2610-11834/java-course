# Guion de video - Lección 2: Variables, tipos de datos y operadores

**Duración sugerida**: 10 a 12 minutos  
**Herramienta**: JDoodle, Replit u OnlineGDB  
**Objetivo**: Que el estudiante aprenda a guardar datos y realizar operaciones simples.

## Estructura del video

### 0:00 - 1:00 | Repaso breve
Recordar que en la lección anterior se mostró texto con `System.out.println`.

### 1:00 - 3:00 | ¿Qué es una variable?
Explica que una variable es como una caja con nombre donde se guarda información.

### 3:00 - 6:00 | Tipos de datos básicos
Mostrar ejemplos:

```java
String name = "Alex";
int age = 15;
double average = 16.5;
boolean likesJava = true;
```

Explica cada tipo con ejemplos cotidianos.

### 6:00 - 9:00 | Operadores y expresiones
Mostrar operaciones simples:

```java
int nextAge = age + 1;
double improvedAverage = average + 1.0;
```

Explicar suma, resta, multiplicación y división de forma básica.

### 9:00 - 11:00 | Práctica rápida
Pedir que cambien los valores por sus propios datos ficticios y ejecuten el programa.

### 11:00 - 12:00 | Cierre
Concluir que los programas pueden almacenar y transformar información.

## Checklist de grabación

- Explicar cada tipo de dato con ejemplos simples.
- No avanzar demasiado rápido.
- Probar el código completo antes de finalizar.
