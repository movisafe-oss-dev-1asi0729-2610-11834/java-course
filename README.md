# Fundamentos de Java para Principiantes

## Resumen del Curso

Este curso introduce a estudiantes de secundaria a los fundamentos de la programación con Java, incluyendo conceptos básicos de programación orientada a objetos. El contenido está organizado de forma progresiva para que cualquier estudiante sin experiencia pueda iniciar desde cero y terminar creando un pequeño programa basado en objetos.

**¡No requiere descargas!** Todas las prácticas se realizan desde el navegador usando herramientas online gratuitas como Replit, JDoodle u OnlineGDB.

**Duración total**: ~60 minutos  
**Público objetivo**: Estudiantes de 12 a 17 años sin experiencia en programación  
**Prerrequisitos**: Ninguno  
**Herramientas necesarias**: Solo un navegador web moderno: Chrome, Firefox, Safari o Edge  
**Repositorio de código fuente**: [https://github.com/usuario/java-fundamentals-course-nombre-equipo](https://github.com/usuario/java-fundamentals-course-nombre-equipo)

## Secuencia de la Lección

### Lección 1: ¿Qué es Java y la programación? (8-10 minutos)

- **Descripción**: Aprende qué es la programación, qué es Java y cómo ejecutar tu primer programa usando un editor online.
- **Enlace**: [Ver la lección](https://youtu.be/PHZHwOWU9-c)
- **Conclusiones clave**: Programar significa dar instrucciones a una computadora. Java permite crear programas organizados y fáciles de entender cuando se empieza paso a paso.
- **Empieza a programar**: [Abrir JDoodle - Hola Mundo](https://www.jdoodle.com/ga/9QiPGcC81d9QpdXLHWAyhQ%3D%3D)
- **Archivos**:
  - Código inicial: [`starter-files/lesson-01/Main.java`](starter-files/lesson-01/Main.java)
  - Código final: [`completed-examples/lesson-01/Main.java`](completed-examples/lesson-01/Main.java)
  - Guion: [`scripts/lesson-01-script.md`](scripts/lesson-01-script.md)

### Lección 2: Variables, tipos de datos y operadores (7 minutos)

- **Descripción**: Aprende a guardar información en Java usando variables, tipos de datos y operaciones básicas.
- **Enlace**: [Ver la lección](https://youtu.be/l0VEy9wLK1o)
- **Conclusiones clave**: Las variables permiten almacenar datos como números, textos y valores verdaderos o falsos. Los operadores permiten realizar cálculos y formar expresiones.
- **Práctica**: [Abrir JDoodle - Variables](https://www.jdoodle.com/online-java-compiler/)
- **Archivos**:
  - Código inicial: [`starter-files/lesson-02/Main.java`](starter-files/lesson-02/Main.java)
  - Código final: [`completed-examples/lesson-02/Main.java`](completed-examples/lesson-02/Main.java)
  - Guion: [`scripts/lesson-02-script.md`](scripts/lesson-02-script.md)

### Lección 3: Estructuras de control: decisiones y bucles (12 minutos)

- **Descripción**: Aprende a tomar decisiones con `if-else` y a repetir instrucciones usando `for` y `while`.
- **Enlace**: [Ver la lección](https://www.youtube.com/watch?v=example-link-3)
- **Conclusiones clave**: Las condicionales permiten elegir qué camino sigue el programa. Los bucles permiten repetir instrucciones sin escribirlas muchas veces.
- **Práctica**: [Abrir OnlineGDB - Condicionales y bucles](https://www.onlinegdb.com/online_java_compiler)
- **Archivos**:
  - Código inicial: [`starter-files/lesson-03/Main.java`](starter-files/lesson-03/Main.java)
  - Código final: [`completed-examples/lesson-03/Main.java`](completed-examples/lesson-03/Main.java)
  - Guion: [`scripts/lesson-03-script.md`](scripts/lesson-03-script.md)

### Lección 4: Métodos, parámetros y entrada de datos (12 minutos)

- **Descripción**: Aprende a crear métodos para reutilizar código, enviar parámetros y leer datos desde la consola con `Scanner`.
- **Enlace**: [Ver la lección](https://youtu.be/iOEm7Hg73eM?si=qZfo1ud7vbWUQPi9)
- **Conclusiones clave**: Los métodos ayudan a ordenar el código. Los parámetros permiten que un método reciba información y la use para realizar una tarea.
- **Práctica**: [Abrir JDoodle - Variables](https://www.jdoodle.com/online-java-compiler/)
- **Archivos**:
  - Código inicial: [`starter-files/lesson-04/Main.java`](starter-files/lesson-04/Main.java)
  - Código final: [`completed-examples/lesson-04/Main.java`](completed-examples/lesson-04/Main.java)
  - Guion: [`scripts/lesson-04-script.md`](scripts/lesson-04-script.md)

### Lección 5: Fundamentos de POO y mini proyecto final (11 minutos)

- **Descripción**: Aprende qué son las clases, objetos, atributos, constructores, métodos, getters y setters. Como cierre, se construye un registro simple de estudiante.
- **Enlace**: [Ver la lección](https://youtu.be/KXsubbNXhOc)
- **Conclusiones clave**: Una clase funciona como un molde y un objeto es un elemento creado desde ese molde. La encapsulación ayuda a proteger los datos usando métodos de acceso.
- **Proyecto final**: [Abrir OnlineGDB - Proyecto POO](https://www.onlinegdb.com/online_java_compiler)
- **Archivos**:
  - Código inicial: [`starter-files/lesson-05/Main.java`](starter-files/lesson-05/Main.java)
  - Código final: [`completed-examples/lesson-05/Main.java`](completed-examples/lesson-05/Main.java)
  - Guion: [`scripts/lesson-05-script.md`](scripts/lesson-05-script.md)
